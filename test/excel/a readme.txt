
基础类型:
	short		短整型		-32768 ~ 32767
	int			整型		-2147483648 ~ 2147483647
	float		单精度		3.40282347E+38F ~  -3.40282347E+38F
	double		双精度		-1.7976931348623157E+308 ~ 1.7976931348623157E+308
	string		字符串类型	也是通用格式(任意格式)类型
	vector2		二维向量	[10.5,10.8]
	vector3		三维向量	[10.5,10.8,0]
	vector4		四维向量	[1,1,1,1]
	
复合类型:
	json		任意json类型{"key":[1,2]}, [{"key":111},{"value":222}]
	array		数组类型	[1,3,4,5]
	object		Table类型	{"key1":10,"key2":20}
	
数组: 应用于所有基础类型
	int[] 		变长数组	[1,2,3,4,...]
	int[2] 		定长数组	[1,2]

	
跨字段组合:
	跨字段数组:	a_字段名_索引 	a_sound_1, a_sound_2  ==> 同等于单字段: int[2]
	跨字段字典: k_键_索引, v_值_索引
				k_item_1, v_item_1  ==> 同等于单字段 {"10001":12,"10002":99}
				不同的是 跨字段字典可以支持数字类型的key, 而单字段不可以
